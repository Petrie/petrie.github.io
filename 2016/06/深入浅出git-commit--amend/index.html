深入浅出'Git commit --amend'

2016年06月23日      18 字 - 1 分钟

Git能够提供悔棋的奥秘在于Git的reset命令。Git commit --amend 可以理解为Git reset 的alias。

Git commit --amend ‘this is amend commit’

等价于

Git reset --soft HEAD^
