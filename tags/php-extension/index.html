PHP扩展开发－自动生成扩展骨架

PHP扩展编译有两种方式

  • 作为一个可装载模块或者DSO（动态共享对象）
  • 静态编译到PHP

静态编译的方式直接和PHP编译到一起，步骤比较简单，但每次变更代码都要编译这个PHP代码，非常耗时，所以这里采用第二种方式
